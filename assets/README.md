# QuizUp+ Logo ve İkonlar

Bu klasör, QuizUp+ uygulamasının tüm görsel varlıklarını içerir.

## Dosya Yapısı

```
assets/
├── logo.svg          # Ana logo (Header ve Landing page için)
├── logo-white.svg    # Beyaz/açık renkli arka planlar için logo versiyonu
├── favicon.svg       # Browser sekmesi ikonu
└── README.md         # Bu dosya
```

## Logo Gereksinimleri

### 1. Ana Logo (`logo.svg`)
- **Kullanım Yeri:** Header (Sidebar), Landing page
- **Önerilen Boyut:** 200x200px veya vektör (SVG)
- **Format:** SVG (tercih edilen) veya PNG
- **Arka Plan:** Şeffaf
- **Renk:** Turuncu tema (#FF5722) ile uyumlu

### 2. Beyaz Logo (`logo-white.svg`)
- **Kullanım Yeri:** Koyu arka planlar (opsiyonel)
- **Önerilen Boyut:** 200x200px veya vektör (SVG)
- **Format:** SVG (tercih edilen) veya PNG
- **Renk:** Beyaz/Açık renkler

### 3. Favicon (`favicon.svg`)
- **Kullanım Yeri:** Browser sekmesi, bookmark
- **Önerilen Boyut:** 64x64px minimum
- **Format:** SVG (tercih edilen), PNG veya ICO
- **Renk:** Basit, tanınabilir

## Dosyaları Nasıl Güncellerim?

1. Logoları bu klasöre yükleyin
2. Dosya adlarını yukarıdaki gibi ayarlayın
3. Git commit ve push yapın
4. Netlify otomatik olarak güncellenecektir

## Mevcut Placeholder Logolar

Şu anda bu klasördeki logolar placeholder (geçici) logolardır.
Kendi logolarınızla değiştirmek için aynı dosya adlarını kullanın.

## Notlar

- SVG formatı tercih edilir (her boyutta keskin görünür)
- PNG kullanıyorsanız en az 200x200px yükleyin
- Dosya boyutunu 500KB altında tutmaya çalışın
- Şeffaf arka plan kullanın
